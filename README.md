# 🏎️ Renta Car Premium — Luxury Experience Medellín

Sitio web oficial interactivo y de alta gama para la reserva y renta de vehículos exclusivos, exóticos y camionetas SUV blindadas en Medellín, Colombia.

## ✨ Características Principales
* **Fondo de Video de Impacto:** Carga inmediata de video cinematográfico en alta definición en la sección de inicio.
* **Sistema Multiidioma Dinámico:** Traducción instantánea en tiempo real sin recargar la página (Español, Inglés, Francés, Chino).
* **Filtros Avanzados y Ordenamiento:** Filtrado por categoría (SUVs / Deportivos) y ordenamiento dinámico por precios en tiempo real.
* **Conversión de Divisas Automática:** Los precios cambian de formato y moneda (COP, USD, EUR, CNY) según el idioma seleccionado.
* **Integración Directa con Canales de Venta:** Enlaces automatizados hacia WhatsApp Business recopilando los datos del formulario de reserva.

## 🛠️ Tecnologías Utilizadas
* HTML5 / CSS3 Avanzado (Variables nativas, Flexbox, CSS Grid).
* JavaScript ES6 (Manipulación dinámica del DOM).
* **GSAP (GreenSock Animation Platform):** Animaciones fluidas de entrada y ScrollTrigger.
* **Particles.js:** Efecto visual premium de partículas flotantes en rojo y blanco.
* **FontAwesome:** Biblioteca de iconos vectoriales.

## 🚀 Despliegue
Diseñado para ser desplegado instantáneamente en entornos estáticos como **GitHub Pages** o Vercel de manera directa.
